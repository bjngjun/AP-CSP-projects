# AP-CSP-projects
A collection of python projects I made in AP CSP 

[atm.py](atm.py)         - A command-line Python script simulating a basic ATM that handles account balances, withdrawals, and deposits.

[create.py](create.py)   - CSV-powered Hangman game with adjustable difficulties and text-based visuals.

[faces.py](faces.py)     - Auto-converter that switches text emoticons into emoji symbols.

[linear.py](linear.py)   - Automated script comparing Binary Search vs Linear Search for a random number.

[pokemon.py](pokemon.py) - Tamagotchi-style trainer sim with Poliwag evolutions and ASCII art.

[rps.py](rps.py)         - Dynamic Rock, Paper, Scissors game keeping real-time win/loss/tie metrics.

[slots.py](slots.py)     - Multi-tier slot machine game featuring symbol payouts and a bulk testing tool.
